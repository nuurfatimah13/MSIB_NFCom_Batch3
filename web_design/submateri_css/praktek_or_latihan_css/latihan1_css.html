Hallo! Saya Fathimah Nuurussa'aadah, seorang mahasiswa semester 5 yang memiliki rasa penasaran tinggi terhadap sesuatu yang dapat membuat saya tertarik. Saya berkuliah di salah satu kampus swasta daerah Depok dan Jakarta Selatan. Nama kampus saya yaitu Sekolah Tinggi Teknologi Terpadu Nurul Fikri. Saya berkuliah mengambil program studi Teknik Informatika.

Sebelumnya, saya lahir di Bogor pada hari Senin tanggal 20 Mei 2002. Dalam keluarga, saya merupakan anak perempuan satu-satunya. Jadi, saya harap agar tidak heran apabila saya tidak seperti anak perempuan kebanyakan dalam segi sikap maupun yang lainnya. Sekedar informasi, saya lebih mudah memahami dan berteman dengan anak laki daripada anak perempuan. Don't judge me!

Saya memiliki sebuah impian terbesar dalam hidup saya yaitu dapat memberangkatkan orangtua saya ke tanah suci Mekkah. Saya memiliki cita-cita yaitu sebagai dosen dan programmer handal yang dapat bermanfaat bagi orang lain. Saya memiliki kebiasaan jika saat gabut maka saya akan mencari pelampiasan seperti editting video, menulis, dan membuat project yang melintas di pikiran saya atau menyibukkan diri dengan hal positif lain. Hal tersebut juga yang akan dijadikan sebagai pelampiasan saat mental saya sedang down.
